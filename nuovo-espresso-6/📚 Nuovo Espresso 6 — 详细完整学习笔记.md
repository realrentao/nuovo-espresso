| -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------- |

### 语法点 — Punti grammaticali

### 1. 不变名词 — I Nomi Invariabili

在意大利语中，部分名词的单复数形式相同，称为**不变名词（nomi invariabili）**。

#### 不变名词分类表

| **类别**          | **说明**  | **例词**                                                                                                                         |
| --------------- | ------- | ------------------------------------------------------------------------------------------------------------------------------ |

## 延伸阅读：Carlo Levi 和 "Cristo si è fermato a Eboli"

**Biografia:** Carlo Levi (1902-1975), scrittore, pittore e antifascista torinese. Nel 1935 fu condannato al confino dal regime fascista e mandato in paesi del Sud Italia (Lucania, oggi Basilicata). Da questa esperienza nacque "Cristo si è fermato a Eboli" (1945).

### 语法点 — Punti grammaticali

### 1. 部分关系代词 — I Pronomi Relativi Partitivi

部分关系代词用于表示"其中……"的概念，在表达数量关系时使用。

#### 结构对照表

| **Struttura**                          | **Esempio**                                                    | **中文**             |
| -------------------------------------- | -------------------------------------------------------------- | ------------------ |
| **di cui + quantità**                  | Ha scritto molti libri, **di cui** quattro a tema scientifico. | 他写了很多书，其中四本是科学主题的。 |

### 2. 假否定 — La Falsa Negazione

假否定是指通过否定其反义词来表达肯定概念的修辞手法。动词本身的意义与想要传达的信息相反。

#### 假否定表达对照表

| **Espressione**                   | **Significato reale**  | **中文**          |
| --------------------------------- | ---------------------- | --------------- |

| **Non è un mistero che...**       | Tutti sanno che...     | 并不神秘 = 众所周知     |
| **Non stupisce che...**           | È normale che...       | 并不奇怪 = 很正常      |
| **Non è obbligato a...**          | È libero di...         | 没有义务 = 可以自由……   |
| **Non è scontato che...**         | È una sorpresa che...  | 并非理所当然 = 令人惊讶   |

| **Non è raro che...**             | È frequente che...     | 并不罕见 = 经常发生     |

### 3. 改变含义的动词：直陈式 vs 虚拟式 — Verbi che cambiano significato

| **Verbo**       | **Con Indicativo**                                                   | **Con Congiuntivo**                                                          |
| --------------- | -------------------------------------------------------------------- | ---------------------------------------------------------------------------- |
| **Prevedere**   | 提前知道（将来时直陈式，有灵主语）—— "Prevedo che l'Italia **vincerà** i mondiali."   | 规定、指出（无灵主语）—— "Il regolamento prevede che gli studenti **vengano** espulsi." |
| **Considerare** | 考虑到—— "**Considera** che dobbiamo essere a casa per cena."           | 假设、设想—— "**Consideriamo** per un momento che lui **abbia** ragione."         |
| **Ammettere**   | 承认—— "Ha **ammesso** che **avevo** ragione io."                      | 假设—— "**Ammettiamo** che Michele **possa** aiutarci."                        |
| **Capire**      | 理解—— "**Capisco** perfettamente che la situazione **è** complicata." | 认为自然—— "**Capisco** che lui **voglia** andarsene da qui."                    |
| **Credere**     | ——                                                                   | 认为（+虚拟式，表示不确定性）—— "**Credo** che lui **abbia** ragione."                     |
| **Pensare**     | ——                                                                   | 认为（+虚拟式，表达主观意见）—— "**Penso** che **sia** giusto."                            |

#### 完整例句 (10+)

| **Italiano**                                                             | **中文**          |
| ------------------------------------------------------------------------ | --------------- |
| **Prevedo** che il progetto **sarà** completato in tempo.                | 我预计项目将按时完成。     |
| La legge **prevede** che i cittadini **abbiano** diritto all'istruzione. | 法律规定公民有受教育的权利。  |
| **Considera** che domani **c'è** sciopero dei treni.                     | 考虑到明天有火车罢工。     |
| **Consideriamo** che lui **abbia** detto la verità.                      | 假设他说的是真话。       |
| **Ammetto** che **avevi** ragione tu.                                    | 我承认你是对的。        |
| **Ammettiamo** che **piova**, cosa facciamo?                             | 假设下雨，我们怎么办？     |
| **Capisco** che la situazione **è** difficile.                           | 我理解情况很困难。       |
| **Capisco** che tu **voglia** andare via.                                | 我理解你想离开（这是自然的）。 |
| **Credo** che Mario **sia** una persona onesta.                          | 我认为Mario是个诚实的人。 |
| **Penso** che **dovremmo** partire subito.                               | 我认为我们应该马上出发。    |

### 1. 副词 "pure" 和 "anzi" — Gli avverbi "pure" e "anzi"

#### **pure** 的用法

| **Uso**                               | **Descrizione** | **Esempio**                                                 |
| ------------------------------------- | --------------- | ----------------------------------------------------------- |
| **Anche**                             | 也，同样            | "Sarà brutto tempo oggi, domani e **pure** domenica."       |
| **Anche se + gerundio (pur)**         | 即使（后跟副动词）       | "**Pur** ammettendo che abbia ragione, non farò come dice." |
| **Esortazione / concessione forzata** | 勉强同意/鼓励         | "Andiamo **pure**, ma dobbiamo tornare presto."             |

#### **anzi** 的用法

| **Uso**              | **Descrizione** | **Esempio**                                      |
| -------------------- | --------------- | ------------------------------------------------ |
| **Modificare**       | 修正前面的话          | "Ti scrivo un sms, **anzi**, ti telefono."       |
| **Rafforzare**       | 加强              | "Hai fatto male, **anzi**, malissimo!"           |
| **Esclamazione**     | 感叹（恰恰相反）        | "Nessun disturbo. **Anzi!** Mi fa piacere!"      |
| **Contrapposizione** | 对比              | "Non è stupido, **anzi**, è molto intelligente." |

#### 完整例句 (10+)

| **Italiano**                                                 | **中文**               |
| ------------------------------------------------------------ | -------------------- |
| Vengo **pure** io alla festa.                                | 我也去派对。               |
| **Pur** essendo stanco, è venuto alla riunione.              | 尽管累了，他还是来开会了。        |
| Fai **pure** come credi.                                     | 你就按你的想法做吧。           |
| Non è antipatico, **anzi**, è molto simpatico.               | 他不讨厌，恰恰相反，他非常友善。     |
| Ti accompagno a casa, **anzi**, ti offro anche un caffè.     | 我送你回家，不仅如此，我还请你喝杯咖啡。 |
| **Anzi!** Tutto il contrario!                                | 恰恰相反！完全相反！           |
| **Pure** ammettendo che sia vero, non cambia niente.         | 即使承认这是真的，也改变不了什么。    |
| Dimmelo **pure**, non ti vergognare.                         | 尽管告诉我，别害羞。           |
| Non solo non mi ha aiutato, **anzi**, mi ha creato problemi. | 他不仅没有帮我，反而给我制造了麻烦。   |
| **Anzi**, scusa, volevo dire un'altra cosa.                  | 哦不，抱歉，我想说的是另一件事。     |

### 2. 连接词 — I Connettivi

连接词是将话语的不同部分连接起来的词或表达。

#### 连接词分类表

| **Categoria**    | **Connettivi**                    | **Esempio**                                                    |
| ---------------- | --------------------------------- | -------------------------------------------------------------- |
| **Aggiuntivi**   | e, anche, inoltre, pure, nonché   | "Ho comprato pane **e** latte, **inoltre** ho preso il burro." |
| **Avversativi**  | ma, però, invece, anzi, tuttavia  | "Volevo uscire, **ma** piove."                                 |
| **Concessivi**   | benché, sebbene, nonostante, pur  | "**Benché** stanco, ha finito il lavoro."                      |
| **Conclusivi**   | quindi, dunque, perciò, pertanto  | "Ho studiato, **quindi** sono preparato."                      |
| **Dichiarativi** | cioè, infatti, ossia, ovvero      | "È italiano, **cioè** di Roma."                                |
| **Temporali**    | quando, mentre, dopo, prima       | "**Quando** arrivi, chiamami."                                 |
| **Causali**      | perché, poiché, siccome, dato che | "**Siccome** piove, resto a casa."                             |
| **Finali**       | affinché, perché, in modo che     | "Studio **affinché** possa capire."                            |

#### 完整例句 (10+)

| **Italiano**                                                     | **中文**              |
| ---------------------------------------------------------------- | ------------------- |
| Ha studiato molto **nonché** lavorato sodo.                      | 他努力学习并且努力工作。        |
| Non mi piace il calcio, **tuttavia** guarderò la partita con te. | 我不喜欢足球，但还是会和你一起看比赛。 |
| Sembrava semplice, **invece** era molto complesso.               | 看起来简单，实际上却很复杂。      |
| Non hai ragione, **anzi**, hai torto marcio.                     | 你没道理，恰恰相反，你大错特错。    |
| Piove a dirotto, **pertanto** rimandiamo la gita.                | 下着倾盆大雨，因此我们推迟远足。    |
| È un medico, **ossia** una persona che cura i malati.            | 他是医生，即治疗病人的人。       |
| **Dato che** sei qui, possiamo parlare.                          | 既然你在这里，我们可以谈谈。      |
| **Affinché** tu capisca, ti farò un esempio.                     | 为了让你理解，我给你举个例子。     |
| Ho fame, **perciò** vado a mangiare.                             | 我饿了，所以我去吃饭。         |
| **Sebbene** piovesse, siamo usciti lo stesso.                    | 尽管下雨了，我们还是出去了。      |

### 3. 感情代词（与格伦理）— Il pronome affettivo (dativo etico)

与格伦理是一种冗余的间接宾语代词（删除后不改变语法性或意义），用于表达情感参与。

#### 人称对照表

| **Pronome**  | **Funzione** | **Esempio**                            |
| ------------ | ------------ | -------------------------------------- |
| **mi / ci**  | 表达说话者的关心/兴趣  | "Che **mi** combini? **Stammi** bene." |
| **ti / vi**  | 让听话者参与       | "All'arrivo chi **ti** incontro?"      |
| **gli / le** | 用于谈论孩子/宠物    | "Giulia non **le** cresce tanto."      |
| **Con ecco** | 与ecco连用      | "**Eccoti** le chiavi!"                |

#### 完整例句 (10+)

| **Italiano**                                     | **中文**                    |
| ------------------------------------------------ | ------------------------- |
| **Stammi** bene, e non fare sciocchezze!         | 好好保重，别做傻事！                |
| Che **mi** combini di bello?                     | 你在搞什么名堂？                  |
| Alla festa, chi **ti** vedo? Maria con un altro! | 在派对上，你猜我看见谁了？Maria和别人在一起！ |
| Il bambino non **mi** mangia niente.             | 这孩子（我的）什么都不吃。             |
| **Eccoti** il libro che cercavi.                 | 给你你要找的书。                  |
| Come **ti** è andata?                            | （你）怎么样了？                  |
| La nonna **le** vuole tanto bene al nipotino.    | 奶奶非常爱小孙子。                 |
| **Eccovi** serviti!                              | （给你们）上菜了！                 |
| **Eccotele** finalmente!                         | 终于给你了！（ecco + ti + le）    |
| Che **ci** fai qui a quest'ora?                  | 这个时间你在这里干什么？              |

### 4. 反身代词与所属关系 — Il riflessivo e il possesso

在提到身体部位或紧密接触的物体时，使用反身代词+定冠词，而不是物主形容词。

| **Corretto**                     | **Sbagliato**                 | **中文**     |

### 2. 亲属关系前的定冠词+物主形容词 — Articolo determinativo con possessivi e parentela

### 语法点 — Punti grammaticali

### 1. 冠词省略 — Omissione dell'articolo

当动词和名词形成一个意义单位时，冠词可以省略。

#### 有冠词 vs 无冠词对照表

| **无冠词 (意义单位)**         | **有冠词 (字面意义)**               |
| ---------------------- | ---------------------------- |
| **fare appello** (求助)  | **fare l'appello** (点名)      |
| **dare spago** (鼓励)    | **dare lo spago** (给绳子)      |
| **prendere nota** (记下) | **prendere la nota** (收处分通知) |
| **tenere conto** (考虑)  | **tenere il conto** (数数/记账)  |
| **dare voce** (让发言)    | **dare la voce** (配音)        |
| **fare colpo** (留下好印象) | **fare il colpo** (盗窃)       |
| **avere fame** (饿)     | _字面义单独存在_                    |
| **volere bene** (爱)    | _字面义单独存在_                    |
| **cercare casa** (找房子) | _字面义单独存在_                    |

### 2. 隐性从句 — Le subordinate implicite

隐性从句是指使用动词的非限定形式（分词、副动词、不定式）替代限定形式（带连词的从句）。

#### 三种形式对照表

| **Forma**           | **Esempio**                                          | **对应显性从句**                                      |
| ------------------- | ---------------------------------------------------- | ----------------------------------------------- |
| **Participio (分词)** | "**Visti** i precedenti, l'imputato è colpevole."    | "**Dopo che sono stati visti** i precedenti..." |
| **Gerundio (副动词)**  | "**Considerando** i precedenti penali..."            | "**Se consideriamo** i precedenti..."           |
| **Infinito (不定式)**  | "**Essere** o non **essere**, questo è il problema." | "Che uno sia o non sia..."                      |

#### Uso assoluto (绝对用法)

当从句主语与主句主语不同时，使用"绝对分词/副动词结构"。

| **Esempio**                                               | **Traduzione**       |
| --------------------------------------------------------- | -------------------- |
| "**Partito** Luigi, non sapemmo più cosa fare."           | Luigi离开后，我们不知道该怎么办了。 |
| "**Essendo** stati informati del pericolo, si fermarono." | 被告知危险后，他们停了下来。       |

#### 完整例句 (10+)

| **Italiano**                                                            | **中文**          |
| ----------------------------------------------------------------------- | --------------- |
| **Considerando** i precedenti penali, l'imputato meritava una condanna. | 考虑到犯罪前科，被告应被判刑。 |
| **Visti** i risultati, possiamo essere soddisfatti.                     | 看到了结果，我们可以满意了。  |
| **Essendo** stato avvisato, non può lamentarsi.                         | 既然已被告知，他不能抱怨。   |
| **Arrivato** a casa, mi sono riposato.                                  | 到家后，我休息了。       |

### 语法点 — Punti grammaticali

### 1. 短语动词 — I verbi frasali

短语动词由动词+小品词（介词或副词）组成，用于改变或加强动词的意义。

#### 分类对照表

| **Tipo**                           | **Esempio**                           | **Significato**        |
| ---------------------------------- | ------------------------------------- | ---------------------- |
| **Significato letterale + luogo**  | "Tira **fuori** la lingua!"           | 伸出（舌头）—— 字面义           |
| **Significato particolare**        | "Ho buttato **giù** due righe."       | 写下（buttare giù = 草草写下） |
| **Valore intensivo (eliminabile)** | "Entrate **(dentro)** ad aspettarlo." | 进去（dentro可省略）          |

#### 常用短语动词表

| **Verbo frasale**  | **Significato** | **Esempio**                               |
| ------------------ | --------------- | ----------------------------------------- |
| **buttare giù**    | 写下 / 使沮丧        | "Ho **buttato giù** un appunto."          |
| **buttare via**    | 扔掉              | "**Butta via** la carta."                 |
| **dare via**       | 赠送              | "**Dà via** i suoi vecchi libri."         |
| **fare fuori**     | 杀死 / 耗尽         | "**Hanno fatto fuori** tutte le scorte."  |
| **andare via**     | 离开 / 消失         | "**Vai via** da qui!"                     |
| **tirare avanti**  | 维持 / 继续         | "**Tiriamo avanti** come possiamo."       |
| **tirare fuori**   | 取出 / 拿出         | "**Tira fuori** la lingua!"               |
| **venire fuori**   | 出来 / 结果是        | "**È venuto fuori** un problema."         |
| **entrare dentro** | 进入              | "**Entrate dentro** a prendere un caffè." |
| **salire su**      | 登上              | "**Sali su** la macchina!"                |

### 3. 反叙法（曲言法）— La litote

反叙法是一种通过否定其反面来缓和或加强语义的修辞手法。

#### 功能对照表

| **Funzione**            | **Esempio**                                 | **Significato reale** |
| ----------------------- | ------------------------------------------- | --------------------- |
| **Sfumare / Attenuare** | "Un ragazzo **nient'affatto stupido**"      | intelligente          |
| **Enfasi**              | "La cena **non era niente male**"           | era buona             |
| **Ironia**              | "Don Abbondio **non era un cuor di leone**" | era un codardo        |
| **Limitazione**         | "Il nuovo collega mi pare **poco sveglio**" | è lento a capire      |
| **Eufemismo**           | "**Diversamente abili**"                    | disabili              |

#### 常用反叙法表达

| **Espressione**           | **Significato reale** |
| ------------------------- | --------------------- |
| **non male**              | abbastanza buono      |
| **nient'affatto stupido** | intelligente          |
| **non poco**              | molto                 |
| **non è raro**            | è frequente           |
| **non è difficile**       | è facile              |
| **non è impossibile**     | è possibile           |
| **non è escluso**         | è possibile           |
| **poco convincente**      | debole, scarso        |
| **poco simpatico**        | antipatico            |
| **poco chiaro**           | confuso, ambiguo      |

| **Italiano**                                                | **中文**         |
| ----------------------------------------------------------- | -------------- |
| Il film **non era niente male**.                            | 这部电影还不错。       |
| È una persona **nient'affatto stupida**.                    | 他是一个一点都不笨的人。   |
| **Non è raro** che lui arrivi in ritardo.                   | 他迟到并不罕见。       |
| **Non è difficile** capire il problema.                     | 理解问题并不难。       |
| **Non è impossibile** che vincano.                          | 他们赢并非不可能。      |
| **Non è escluso** che partiamo domani.                      | 不排除我们明天出发。     |
| La sua spiegazione è stata **poco chiara**.                 | 他的解释不太清楚。      |

### Strategie per rendere un testo orale in scritto

1. **Eliminare ripetizioni e pause** — 删除重复和停顿
2. **Strutturare in paragrafi** — 分段组织
3. **Usare connettivi formali** — 使用正式连接词
4. **Eliminare interiezioni** — 删除感叹词
5. **Completare frasi incomplete** — 补全不完整的句子
6. **Sostituire deittici con riferimenti espliciti** — 用明确指代替换指示词
