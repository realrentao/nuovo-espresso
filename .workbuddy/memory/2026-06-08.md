# 2026-06-08 工作记录

## Nuovo Espresso 1 交互式 HTML 课程设计（新增）
基于 PDF《Nuovo Espresso 1 Corso di italiano Libro dello studente e esercizi》设计，参考 DIECI A1 配色/布局/音频体系。

### 文件结构 (nuovo-espresso/)
- `css/ne-styles.css` - 橙色主题样式（红/橙/蓝配色）
- `js/ne-audio-player.js` - 音频播放器
- `audio/ne-audio-data.js` - 103个意大利语女声 base64 音频数据
- `index.html` - 课程主页（10单元卡片网格）
- `unita-1.html` - Primi contatti
- `unita-2.html` - Io e gli altri
- `unita-3.html` - Buon appetito!
- `unita-4-6.html` - Tempo libero + In albergo + In giro per l'Italia
- `unita-7-10.html` - Andiamo in vacanza + Sapori d'Italia + Vita quotidiana + La famiglia

### 功能特性
- 意大利语女声（it-IT-IsabellaNeural）预录音频
- 交互式词汇闪卡（点击发音）
- 语法表格、填空练习、配对练习、选择题测验
- 对话实战、主题标签导航
- 滚动进度条、渐入动画
