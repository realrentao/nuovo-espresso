2. Se avessi saputo prima... — 如果我早知道的话... · p.147, es.4

Completa con il periodo ipotetico dell'irrealtà.（用虚拟现实条件句完成。）

Se avessi saputo prima, sarei venuto in metropolitana.如果我早知道，我就坐地铁来了。
Se fosse venuto, ne sarei stata felice.如果他来了，我会很高兴的。
Se avessi mangiato di meno, non starei così male.如果我少吃点，现在就不会这么难受了。

3. Se io fossi... — 如果我是... · p.148, es.5

Completa le frasi con il congiuntivo imperfetto.（用虚拟式未完成时完成句子。）

Se io fossi un artista, dipingerei quadri bellissimi.如果我是一位艺术家，我会画美丽的画。
Se lui fosse più gentile, tutti lo amerebbero.如果他更友善，大家都会喜欢他。
Se noi avessimo più tempo, viaggeremmo di più.如果我们有更多时间，我们会更多旅行。

📚 Grammatica · 语法点

1. Il periodo ipotetico dell'irrealtà（虚拟现实条件句）

表达过去不可能实现的假设。

结构: se + congiuntivo trapassato + condizionale passato

se从句 (congiuntivo trapassato)主句 (condizionale passato)
se avessi avutoavrei avuto
se fossi statosarei stato
se avessi fattoavrei fatto
se fosse venutosarebbe venuto

例句：

Se avessi saputo prima, sarei venuto in metropolitana.如果我早知道，我就坐地铁来了。
Se fosse venuto, ne sarei stata felice.如果他来了，我会很高兴的。
Se avessi mangiato di meno, non starei così male.如果我少吃点，现在就不会这么难受了。
Se Franco non fosse pessimista e avaro, lo troverei simpatico.如果Franco不是悲观和吝啬，我会觉得他可爱。
Se avessi vinto alla lotteria, avrei comprato una casa.如果我中了彩票，我就买房子了。
Se mi avesse telefonato, sarei andato.如果他给我打电话了，我就去了。
Se avessi studiato di più, avrei superato l'esame.如果我多学习了，我就通过考试了。
Se fossi nato in Italia, parlerei italiano meglio.如果我出生在意大利，我的意大利语会说更好。

2. Congiuntivo trapassato（虚拟式愈过去时）

构成: avere/essere的虚拟式未完成时 + 过去分词

主语avere (cong. imperfetto)essere (cong. imperfetto)
ioavessi parlatofossi andato/a
tuavessi parlatofossi andato/a
lui/lei/Leiavesse parlatofosse andato/a
noiavessimo parlatofossimo andati/e
voiaveste parlatofoste andati/e
loroavessero parlatofossero andati/e

例句：

Se avessi avuto tempo, ti avrei chiamato.如果我有时间，我就给你打电话了。
Se fossi stato presente, avrei aiutato.如果我在场，我就帮忙了。
Se avesse studiato, avrebbe superato.如果他学习了，他就通过了。
Se avessimo saputo, saremmo venuti.如果我们知道，我们就来了。
Se aveste telefonato, avremmo risposto.如果你们打电话了，我们就接了。
Se avessero chiesto, avrei spiegato.如果他们问了，我就解释了。
Se fossi uscito prima, non avrei perso il treno.如果我早点出去，就不会错过火车了。
Se non avesse piovuto, saremmo andati al mare.如果没有下雨，我们就去看海了。

3. Periodo ipotetico — 三种条件句总结

类型结构含义
现实条件句se + 直陈式现在/将来可能实现
可能条件句se + 虚拟式未完成时不太可能实现
虚拟条件句se + 虚拟式愈过去时不可能实现（过去）

例句对比：

Se studi, superi l'esame.如果你学习，你会通过考试。（很可能）
Se studiassi, superresti l'esame.如果你学习，你会通过考试。（不太可能）
Se avessi studiato, avresti superato l'esame.如果你学习了，你就通过考试了。（已经过去）
