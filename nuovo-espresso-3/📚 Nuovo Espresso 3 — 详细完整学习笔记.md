### 语法点

#### 1. Il trapassato prossimo（近过去完成时）

**定义**: Trapassato prossimo（近过去完成时）用于表示在另一个过去动作之前已经发生的动作，即"过去的过去"。

**构成**: avere/essere的未完成过去时(imperfetto) + 过去分词

**变位表格**:

| **主语**      | **avere (imperfetto)** | **essere (imperfetto)** |
| ----------- | ---------------------- | ----------------------- |
| io          | avevo                  | ero                     |
| tu          | avevi                  | eri                     |
| lui/lei/Lei | aveva                  | era                     |
| noi         | avevamo                | eravamo                 |
| voi         | avevate                | eravate                 |
| loro        | avevano                | erano                   |

**例句**:

1. **Avevo studiato l'italiano prima di venire in Italia.** — 在来意大利之前我学过意大利语。
2. **Quando sono arrivato, lui era già partito.** — 当我到达时，他已经出发了。
3. **Ero stanca perché avevo dormito poco.** — 我很累因为我睡得少。
4. **Avevamo mangiato prima di andare al cinema.** — 我们在去看电影之前已经吃过了。
5. **Lei aveva lavorato in Italia per cinque anni prima di tornare in patria.** — 她在回国之前在意大利工作了五年。
6. **Ero felice perché avevo superato l'esame.** — 我很高兴因为我通过了考试。
7. **Avevamo finito i compiti quando è arrivato l'insegnante.** — 当老师来的时候我们已经完成了作业。
8. **Ti avevo detto di venire presto!** — 我之前告诉过你要早点来！

**使用场景**:

* 用于描述在过去某个时间点之前已经完成的动作
* 与quando, prima di, dopo che等连词连用
* 在复合句中表示"过去的过去"

**例句对比**:

| **Trapassato prossimo**               | **Passato prossimo**           | **中文翻译**                  |
| ------------------------------------- | ------------------------------ | ------------------------- |
| Avevo mangiato prima di uscire.       | Ho mangiato e poi sono uscito. | 我出门前吃过了。/ 我吃了然后出门了。       |
| Era già partito quando sono arrivato. | Sono arrivato e poi è partito. | 当我到的时候他已经出发了。/ 我到了然后他出发了。 |

***

#### 2. Prima di + infinito（不定式前置）

**定义**: Prima di + 不定式表示在主要动作之前发生的动作。

**结构**: prima di + 不定式动词

**例句**:

1. **Prima di uscire, ho chiuso la porta.** — 出门之前，我关了门。
2. **Prima di studiare l'italiano, ho imparato lo spagnolo.** — 在学意大利语之前，我学了西班牙语。
3. **Devi chiamarmi prima di partire.** — 你出发前必须给我打电话。
4. **Prima di andare a letto, leggo sempre un po'.** — 上床睡觉之前，我总是读一会儿书。
5. **Prima di rispondere, pensa bene.** — 回答之前，好好想想。
6. **Ha fatto molti progressi prima di raggiungere questo livello.** — 在达到这个水平之前，他取得了很大的进步。
7. **Prima di viaggiare in Italia, ho studiato l'italiano per due anni.** — 在去意大利旅行之前，我学了两年意大利语。
8. **Chiudi le finestre prima di andare via.** — 离开之前把窗户关上。

**注意**:

* Prima di后面必须用不定式，不能用直陈式
* Prima di主要表示时间上的先后
* 与prima che (后面用虚拟式) 区分

#### 3. Il verbo dovere per esprimere un'ipotesi（dovere表达假设）

**定义**: Dovere在直陈式现在时或条件式中可用来表达推测、假设。

**用法**:

| **形式**                    | **含义**        | **例句**                                            |
| ------------------------- | ------------- | ------------------------------------------------- |
| deve + infinito           | 很可能...（强烈推测）  | Deve essere ammalato, non è venuto. 他肯定病了，他没来。    |
| dovrebbe + infinito       | 可能...（较弱推测）   | Dovrebbe essere a casa adesso. 他现在可能在家。           |
| doveva + infinito         | 当时应该...（过去推测） | Doveva essere tardi quando è uscito. 他出去的时候肯定很晚了。 |
| avrebbe dovuto + infinito | 本应该...（遗憾）    | Avrebbe dovuto studiare di più. 他本应该多学习。          |

**例句**:

1. **La grammatica dovrebbe essere lì.** — 语法书可能在那儿。
2. **Dovrebbe essere andato a casa.** — 他可能回家了。
3. **Deve aver preso il treno delle 8:00.** — 他肯定坐了8点的火车。
4. **Doveva essere molto stanco ieri sera.** — 昨晚他肯定很累。
5. **Avrebbe dovuto studiare di più per l'esame.** — 他本应该为考试多学习。
6. **Dovrebbe piovere presto, guarda le nuvole.** — 看着云，可能很快要下雨了。
7. **Marco deve aver dimenticato l'appuntamento.** — Marco肯定忘记了约会。
8. **Avrei dovuto dirtelo prima.** — 我应该早点告诉你的。

**注意**:

* deve/dovrebbe表达推测程度不同
* 在正式书面语中更常用
* 与直陈式结合表达确定的推测

#### 5. Il prefisso negativo in-（否定前缀in-）

**定义**: 形容词前加in-前缀表示否定意义，相当于"non + 形容词"。

**规则**:

| **规则** | **原词**      | **否定词**       | **例子**       |
| ------ | ----------- | ------------- | ------------ |
| 一般情况   | capace      | incapace      | 有能力的 → 无能力的  |
| 以b/p开头 | credibile   | incredibile   | 可信的 → 不可信的   |
| 以m开头   | morale      | immorale      | 道德的 → 不道德的   |
| 以其他音开头 | logico      | illogico      | 合逻辑的 → 不合逻辑的 |
| 以r开头   | ragionevole | irragionevole | 合理的 → 不合理的   |

**常见否定形容词**:

| **形容词**      | **否定形容词**      | **中文翻译**     |
| ------------ | -------------- | ------------ |
| capace       | incapace       | 有能力的 → 无能力的  |
| credibile    | incredibile    | 可信的 → 不可信的   |
| possibile    | impossibile    | 可能的 → 不可能的   |
| logico       | illogico       | 合逻辑的 → 不合逻辑的 |
| paziente     | impaziente     | 有耐心的 → 没耐心的  |
| regolare     | irregolare     | 规则的 → 不规则的   |
| responsabile | irresponsabile | 负责任的 → 不负责任的 |
| risolvibile  | irrisolvibile  | 可解决的 → 不可解决的 |
| tollerabile  | intollerabile  | 可容忍的 → 不可容忍的 |
| visibile     | invisibile     | 可见的 → 不可见的   |
| bevibile     | imbevibile     | 可饮用的 → 不可饮用的 |
| legale       | illegale       | 合法的 → 非法的    |

**例句**:

1. **È una persona capace.** → **È una persona incapace.** — 他是一个有能力的人。→ 他是一个没有能力的人。
2. **Si tratta di una storia credibile.** → **Si tratta di una storia incredibile.** — 这是一个可信的故事。→ 这是一个不可信的故事。
3. **È stato proprio un lavoro utile.** → **È stato proprio un lavoro inutile.** — 这真是一项有用的工作。→ 这真是一项无用的工作。
4. **Questo caffè ti sembra bevibile?** → **Questo caffè è imbevibile!** — 这咖啡你觉得能喝吗？→ 这咖啡没法喝！
5. **È un discorso morale.** → **È un discorso immorale.** — 这是一篇道德的演讲。→ 这是一篇不道德的演讲。
6. **Questa è una conclusione prevista.** → **Questa è una conclusione imprevista.** — 这是一个预料之中的结论。→ 这是一个出乎意料的结论。
7. **La tua proposta è ragionevole.** → **La tua proposta è irragionevole.** — 你的建议是合理的。→ 你的建议是不合理的。
8. **È un comportamento paziente.** → **È un comportamento impaziente.** — 这是一种耐心的行为。→ 这是一种不耐烦的行为。

### Caffè Culturale 1 — Lingue e dialetti in Italia（意大利的语言和方言）

**页码**: p.18

#### 意大利语的多样性

意大利不仅有标准意大利语（l'italiano standard），还有许多地区方言（dialetti）和少数族裔语言。

**关键信息**:

| **项目**    | **内容**              |
| --------- | ------------------- |
| 标准意大利语使用者 | 约85%的意大利人           |
| 方言使用者     | 约15%的意大利人在家使用方言     |
| 少数民族语言    | 斯洛文尼亚语、克罗地亚语、德语、法语等 |
| 地区变体      | 20个大区各有不同的方言特点      |

**意大利语与方言的关系**:

1. **地理分布**: 意大利南北方言差异巨大
   * 北部方言：受伦巴第语、威尼托语影响
   * 南部方言：受那不勒斯语、西西里语影响
   * 中部：更接近标准意大利语
2. **语言特点**:
   * 意大利语是一种综合语言（lingua sintetica）
   * 方言保留了更多古老的拉丁语特征
   * 那不勒斯方言有独特的语法结构
3. **现代趋势**:
   * 年轻人更多使用标准意大利语
   * 社交媒体促进了意大利语的统一
   * 方言在家庭和传统活动中保留

**相关词汇**:

| **意大利语**                 | **中文翻译** |
| ------------------------ | -------- |
| il dialetto              | 方言       |
| la lingua standard       | 标准语言     |
| il vernacolo             | 方言，土话    |
| l'italiano regionale     | 地区意大利语   |
| la minoranza linguistica | 语言少数群体   |

**文化注释**:

* 意大利学校使用标准意大利语教学
* 很多意大利人能说多种语言（意大利语+方言+外语）
* 方言中有些词汇被吸收到标准意大利语中

### 语法点

#### 1. Il condizionale passato（过去条件式）

**定义**: Condizionale passato（过去条件式）用于表达过去未能实现的愿望或本应该发生但没有发生的事情。

**构成**: avere/essere的条件式现在时 + 过去分词

| **io**      | **avrei** | **sarei** |
| ----------- | --------- | --------- |
| tu          | avresti   | saresti   |
| lui/lei/Lei | avrebbe   | sarebbe   |
| noi         | avremmo   | saremmo   |
| voi         | avreste   | sareste   |
| loro        | avrebbero | sarebbero |

**例句**:

1. **Avrebbe potuto aprire una clinica privata (ma non l'ha aperta).** — 他本可以开一家私人诊所（但他没有开）。
2. **Sarebbe stato meglio costruire un parco (ma non l'hanno costruito).** — 建一个公园会更好（但他们没有建）。
3. **Avrei voluto parlarti, ma non ti ho trovato.** — 我本想和你谈谈，但我没找到你。
4. **Sarei venuto volentieri, ma ero malato.** — 我本想来的，但我病了。
5. **Avrebbero dovuto avvertirci prima.** — 他们应该提前通知我们的。
6. **Saresti potuto essere più gentile.** — 你本可以更客气一点。
7. **Avrei mangiato di meno, ma avevo fame.** — 我本应该少吃点，但我饿了。
8. **Sarebbe stato bello se foste venuti.** — 如果你们来了就好了。

**与新闻语言**: 在新闻报道中，条件式过去时用于表达不确定的信息：

* **L'uomo avrebbe rubato due milioni di euro.** — 据说此人偷了两百万欧元。
* **Il politico avrebbe accettato una tangente.** — 据称这位政治家收了贿赂。

#### 2. Le particelle pronominali ci e ne（ci和ne小品词）

**Ci的用法**:

| **功能**     | **含义** | **例句**                                                     |
| ---------- | ------ | ---------------------------------------------------------- |
| 代替con + 名词 | 和...一起 | Vado al cinema con i miei amici. → Ci vado. 我和朋友去看电影。→ 我去。 |
| 表示"这里/那里"  | 地点     | Siamo in ufficio. → Ci siamo. 我们在办公室。→ 我们在这儿。              |
| 代替a + 问题   | 关于某个问题 | Penso al problema. → Ci penso. 我想那个问题。→ 我来想。               |

**Ne的用法**:

| **功能**    | **含义**  | **例句**                                                  |
| --------- | ------- | ------------------------------------------------------- |
| 代替di + 名词 | 关于...   | Ho parlato del viaggio. → Ne ho parlato. 我谈到了旅行。→ 我谈过了。 |
| 表示数量      | ...中的一些 | Ho tre fratelli. → Ne ho tre. 我有三个兄弟。→ 我有仨。             |
| 代替di + 内容 | ...中的   | Prendo del caffè. → Ne prendo. 我喝点咖啡。→ 我喝点。             |

**例句**:

1. **Guadagnavo anche qualcosa, e ci (= con i soldi guadagnati) pagavo gli studi.** — 我也赚了些钱，用这些钱付学费。
2. **Che ne (= di qualcosa che ho detto) pensate?** — 你们怎么想（我说的）？
3. **Andiamo al cinema?** — No, grazie, non ne ho voglia.\*\* — 我们去看电影吗？— 不，谢谢，我没兴趣。
4. **Quanti libri hai letto quest'anno?** — Ne ho letti dieci.\*\* — 今年你读了多少本书？— 我读了十本。
5. **Sei mai stato a Roma?** — Sì, ci sono stato due volte.\*\* — 你去过罗马吗？— 是的，我去过两次。
6. **Ha telefonato Laura?** — Sì, ci ha parlato.\*\* — Laura打电话了吗？— 是的，她通过话了。
7. **Devi parlare con il professore.** — Sì, ne parlerò domani.\*\* — 你得和教授谈谈。— 是的，我明天谈。
8. **Hai bisogno di soldi?** — No, ne ho abbastanza.\*\* — 你需要钱吗？— 不，够了。

#### 4. I pronomi possessivi（所有格代词）

**定义**: 所有格代词代替名词，并带有所有格意义。

**形式**:

| **主语**      | **单数阳性**  | **单数阴性**  | **复数阳性** | **复数阴性**  |
| ----------- | --------- | --------- | -------- | --------- |
| io          | il mio    | la mia    | i miei   | le mie    |
| tu          | il tuo    | la tua    | i tuoi   | le tue    |
| lui/lei/Lei | il suo    | la sua    | i suoi   | le sue    |
| noi         | il nostro | la nostra | i nostri | le nostre |
| voi         | il vostro | la vostra | i vostri | le vostre |
| loro        | il loro   | la loro   | i loro   | le loro   |

**例句**:

1. **È Sua questa Punto rossa?** — Sì, è **mia**. — 这辆红色Punto是您的吗？— 是的，是我的。
2. **Di chi è quest'ombrello?** — È **mio**. — 这把伞是谁的？— 是我的。
3. **Prestami la tua bicicletta. La mia (bicicletta) si è rotta.** — 把你的自行车借我。我的坏了。
4. **Il mio corso è molto interessante. Anche il tuo (corso)?** — 我的课程很有趣。你的也是吗？
5. **Mia sorella si è laureata. E la tua?** — 我姐姐毕业了。你的呢？
6. **È suo questo libro?** — 这本书是您的吗？
7. **I nostri genitori sono in vacanza.** — 我们的父母在度假。
8. **Ho perso i miei occhiali.** — 我丢了我的眼镜。

**注意**:

* 所有格代词前面必须有定冠词（或介词）
* 与所有格形容词区别：形容词修饰名词，代词代替名词
* 特殊用法：per colpa mia/sua, in vita mia, per conto mio等

### 语法点

#### 1. Il congiuntivo passato（虚拟式过去时）

**定义**: Congiuntivo passato（虚拟式过去时）用于表达过去的观点、情感、怀疑等，常跟在credere, pensare, sperare, temere等动词后面。

**构成**: avere/essere的虚拟式现在时 + 过去分词

| **主语**      | **avere (congiuntivo presente)** | **essere (congiuntivo presente)** |
| ----------- | -------------------------------- | --------------------------------- |
| io          | abbia                            | sia                               |
| tu          | abbia                            | sia                               |
| lui/lei/Lei | abbia                            | sia                               |
| noi         | abbiamo                          | siamo                             |
| voi         | abbiate                          | siate                             |
| loro        | abbiano                          | siano                             |

**例句**:

1. **Credo che sia partito.** — 我相信他已经出发了。
2. **Penso che l'abbiano assunto come responsabile.** — 我认为他们已经聘用了负责人。
3. **Ho paura che questo documento sia sbagliato.** — 我担心这份文件是错的。
4. **Spero che abbiate capito la lezione.** — 我希望你们已经理解了这节课。
5. **Non credo che sia venuto ieri.** — 我不相信他昨天来了。
6. **È possibile che abbiano sbagliato numero.** — 他们可能打错号码了。
7. **Temo che non abbiamo avuto il tempo.** — 我担心我们没有时间。
8. **Suppongo che sia già finito.** — 我推测已经结束了。

**使用规则**:

| **主句动词时态** | **从句用时态**              |
| ---------- | ---------------------- |
| 现在时/将来时    | congiuntivo passato    |
| 过去时        | congiuntivo trapassato |

***

#### 2. Ripasso del congiuntivo（虚拟式复习）

**何时使用虚拟式**:

| **类型** | **例句**                                                  |
| ------ | ------------------------------------------------------- |
| 表达观点   | Penso che..., Credo che..., Secondo me...               |
| 表达情感   | Sono felice che..., Ho paura che..., Mi dispiace che... |
| 表达欲望   | Voglio che..., Desidero che..., Spero che...            |
| 表达不确定  | Può darsi che..., È possibile che...                    |
| 无人称结构  | È importante che..., È necessario che...                |

**例句**:

1. **Credo che Roberto sia andato in Cina.** — 我相信Roberto已经去中国了。
2. **Ho l'impressione che tu sia un po' stanco.** — 我感觉你有点累。
3. **L'importante è che tu sia soddisfatto.** — 重要的是你满意。
4. **Non credo che lui lo sappia.** — 我不相信他知道这件事。
5. **È necessario che tutti partecipino.** — 所有人参加是必要的。
6. **Speriamo che tutto vada bene.** — 希望一切顺利。
7. **Temo che sia troppo tardi.** — 我担心太晚了。
8. **Sono contenta che siate venuti.** — 你们来了我很高兴。

***

#### 3. La concordanza dei tempi e dei modi (I)（时态和语气一致规则 I）

**规则**:

| **主句时态**                  | **从句时态**                  |
| ------------------------- | ------------------------- |
| 现在时 → passato prossimo    | 从句用congiuntivo passato    |
| 现在时 → presente            | 从句用congiuntivo presente   |
| 过去时 → trapassato prossimo | 从句用congiuntivo trapassato |
| 过去时 → imperfetto          | 从句用congiuntivo imperfetto |

**例句**:

1. **Ho paura che le cose possano peggiorare.** (= in questi giorni o nel prossimo futuro) — 我担心情况可能会恶化。（在当下或不久的将来）
2. **Può darsi che sia già partito.** (= prima, nel passato) — 他可能已经出发了。（在更早的过去）
3. **Credevo che lui fosse partito.** — 我以为他已经出发了。（主句是过去时）
4. **Pensavo che voi lo sapeste.** — 我以为你们知道。

***

#### 4. A patto che, purché, a condizione che + congiuntivo（条件连词+虚拟式）

**定义**: 这些连词引导条件从句，需要用虚拟式。

**连词对比**:

| **连词**           | **含义**  | **用法** |
| ---------------- | ------- | ------ |
| a patto che      | 条件是...  | 提出条件   |
| purché           | 只要...   | 强调唯一条件 |
| a condizione che | 在...条件下 | 正式条件   |
| a meno che       | 除非...   | 否定条件   |
| prima che        | 在...之前  | 时间     |

**例句**:

1. **È molto bello, a patto che ti piacciano i gialli.** — 很好，只要你喜欢黄色。
2. **Ti presto i miei libri, purché tu me li riporti.** — 我把书借给你，只要你还我。
3. **Posso aiutarti, a condizione che tu mi aiuti anche tu.** — 我可以帮你，条件是你也帮我。
4. **Andremo al mare, a meno che non piova.** — 我们会去海边，除非下雨。
5. **Devi finire prima che io vada via.** — 在我走之前你必须完成。
6. **Verrò, a patto che non sia troppo tardi.** — 我会来，只要不是太晚。
7. **Ti do il permesso, purché tu prometta di essere gentile.** — 我允许你，只要你答应会客气。
8. **Ti aiuto, a condizione che tu faccia la tua parte.** — 我帮你，条件是你也做你那份。

***

#### 5. Il suffisso -accio（后缀-accio，表示贬义）

**定义**: -accio后缀给名词添加负面、贬义的含义。

**例词**:

| **原词**        | **+ -accio**          | **中文翻译** |
| ------------- | --------------------- | -------- |
| il film       | il filmaccio          | 烂片       |
| la giornata   | la giornataccia       | 糟糕的一天    |
| il tempo      | il tempaccio          | 坏天气      |
| la partita    | la partitaccia        | 糟糕的比赛    |
| il ragazzo    | il ragazzaccio        | 坏男孩      |
| il lavoro     | il lavoraccio         | 苦差事      |
| la situazione | la situazione pessima | 糟糕的情况    |
| il gatto      | il gattaccio          | 恶猫       |

**例句**:

1. **Luigi mi ha fatto un gestaccio!** — Luigi做了一个很粗鲁的手势！
2. **Ho passato una nottataccia.** — 我度过了糟糕的一晚。
3. **Non dire le parolacce!** — 别说脏话！
4. **Che tempaccio oggi!** — 今天天气真差！
5. **Abbiamo visto un filmaccio al cinema.** — 我们在电影院看了一部烂片。
6. **Che partitaccia! Non l'ho vista neanche per un minuto.** — 这比赛真烂！我一分钟都没看。

***

#### 6. Gli avverbi in -mente（-mente副词）

**定义**: 大多数副词由形容词的阴性单数形式加-mente构成。

**构成规则**:

| **形容词**   | **阴性形式**  | **副词**         |
| --------- | --------- | -------------- |
| effettivo | effettiva | effettivamente |
| probabile | probabile | probabilmente  |
| certo     | certa     | certamente     |
| vero      | vera      | veramente      |
| speciale  | speciale  | specialmente   |
| naturale  | naturale  | naturalmente   |
| recente   | recente   | recentemente   |
| chiaro    | chiara    | chiaramente    |

**特殊情况**:

| **形容词** | **副词**      | **说明**   |
| ------- | ----------- | -------- |
| veloce  | velocemente | 以-e结尾直接加 |
| grande  | grandemente | 不规则变化    |
| nuovo   | nuovamente  | 不规则变化    |

**例句**:

1. **Effettivamente è strano!** — 确实很奇怪！
2. **Probabilmente hai confuso il numero.** — 你可能搞错了号码。
3. **Certamente verrò alla festa.** — 我肯定会来派对。
4. **Naturalmente, ognuno ha la sua opinione.** — 当然，每个人都有自己的观点。
5. **Veramente non lo sapevo.** — 我真的不知道。
6. **Recentemente ho comprato una casa.** — 最近我买了一栋房子。
7. **Si è comportata gentilmente con tutti.** — 她对每个人都很亲切。
8. **Si muoveva lentamente nel corridoio.** — 他在走廊里慢慢移动。

### 语法点

#### 1. Il congiuntivo imperfetto（虚拟式未完成时）

**定义**: Congiuntivo imperfetto（虚拟式未完成时）用于表达与现在或过去事实相反的假设、愿望等。

**构成**: 动词直陈式简单将来时或现在时词根 + 虚拟式词尾

| **主语**      | **-are动词** | **-ere动词** | **-ire动词** |
| ----------- | ---------- | ---------- | ---------- |
| io          | parlassi   | vendessi   | partissi   |
| tu          | parlassi   | vendessi   | partissi   |
| lui/lei/Lei | parlasse   | vendesse   | partisse   |
| noi         | parlassimo | vendessimo | partissimo |
| voi         | parlaste   | vendeste   | partiste   |
| loro        | parlassero | vendessero | partissero |

**例句**:

1. **Vorrei che tu mi aiutassi.** — 我希望你帮帮我。
2. **Se avessi tempo, verrei volentieri.** — 如果我有时间，我很愿意来。
3. **Era importante che lui fosse presente.** — 他在场很重要。
4. **Dubitavo che loro lo sapessero.** — 我怀疑他们知道这件事。
5. **Se non piovesse, usciremmo.** — 如果不下雨我们就出去。
6. **Mi sembrava che tu fossi stanco.** — 我觉得你累了。
7. **È necessario che voi facciate i compiti.** — 你们做作业是必要的。
8. **Non credevo che fosse così difficile.** — 我不相信这么难。

**不规则动词**:

| **原形** | **io**  | **lui/lei** | **noi**   | **loro**  |
| ------ | ------- | ----------- | --------- | --------- |
| essere | fossi   | fosse       | fossimo   | fossero   |
| avere  | avessi  | avesse      | avessimo  | avessero  |
| fare   | facessi | facesse     | facessimo | facessero |
| dire   | dicessi | dicesse     | dicessimo | dicessero |

***

#### 2. Come se + congiuntivo（好像...）

**定义**: Come se引导的从句用虚拟式未完成时，表示与现实相反的比较。

**结构**: come se + congiuntivo imperfetto

**例句**:

1. **Ti guarda come se non l'avessi mai visto.** — 他看你的眼神好像从没见过你一样。
2. **Parlo come se sapessi tutto.** — 我说话的样子好像什么都知道。
3. **Si comporta come se fosse il padrone.** — 他的行为好像他是老板。
4. **Mi sento come se fossi a casa.** — 我感觉像在家里一样。
5. **Guarda come se non capisse nulla.** — 他看起来好像什么都不懂。
6. **Sembra come se piovesse sempre.** — 好像总是在下雨。
7. **Stai come se non avessi mangiato da giorni.** — 你看起来像好几天没吃东西。
8. **Mi tratta come se fossi un bambino.** — 他把我当小孩一样对待。

***

#### 3. Il discorso indiretto（间接引语）

**定义**: 间接引语用于转述别人的话，需要改变人称、时态和地点/时间副词。

**规则**:

| **直接引语**           | **间接引语**            |
| ------------------ | ------------------- |
| presente →         | imperfetto          |
| passato prossimo → | trapassato prossimo |
| imperfetto →       | imperfetto          |
| futuro →           | condizionale        |
| imperativo →       | di + infinito       |

**人称变化**:

| **io**  | **→** | **lui/lei** |
| ------- | ----- | ----------- |
| noi     | →     | loro        |
| mio/sua | →     | suo         |
| questo  | →     | quello      |
| qui     | →     | lì          |

**时间和地点变化**:

| **qui/qua**           | **→** | **lì/là**                   |
| --------------------- | ----- | --------------------------- |
| oggi                  | →     | quel giorno                 |
| ieri                  | →     | il giorno prima/l'indomani? |
| domani                | →     | il giorno dopo              |
| adesso/ora            | →     | allora                      |
| la settimana prossima | →     | la settimana dopo           |

**例句**:

1. **"Vengo domani."** → Ha detto che **viene domani** / che **sarebbe venuto il giorno dopo**. — 他说明天来。
2. **"Sono stanco."** → Ha detto che **era stanco**. — 他说他累了。
3. **"Andrò a Roma."** → Ha detto che **sarebbe andato a Roma**. — 他说他将去罗马。
4. **"Chiudi la porta!"** → Mi ha detto di **chiudere la porta**. — 他让我关门。
5. **"L'ho visto ieri."** → Ha detto che **l'aveva visto il giorno prima**. — 他说他前一天看到的。
6. **"Vivo qui da dieci anni."** → Ha detto che **viveva lì da dieci anni**. — 他说他在那住了十年。
7. **"Telefonerò domani."** → Ha promesso che **avrebbe telefonato il giorno dopo**. — 他答应第二天打电话。
8. **"Non lo so."** → Ha detto che **non lo sapeva**. — 他说他不知道。

***

#### 4. La costruzione "che io sappia"（据我所知）

**定义**: 这是一个固定表达，用于表示不确定的信息。

**例句**:

1. **Che io sappia, è uno scrittore molto bravo.** — 据我所知，他是一位很优秀的作家。
2. **Che io sappia, non ci sono problemi.** — 据我所知，没有问题。
3. **Che io sappia, non è ancora partito.** — 据我所知，他还没出发。
4. **A me risulta che sia vero.** — 据我了解，这是真的。

***

#### 5. La forma passiva con essere e con venire（被动语态：essere和venire）

**定义**: 被动语态由essere或venire的适当形式 + 过去分词构成。

**essere被动语态**:

| **时态** | **构成**      |
| ------ | ----------- |
| 现在时    | è + 过去分词    |
| 过去时    | era + 过去分词  |
| 将来时    | sarà + 过去分词 |

**venire被动语态**:

| **时态** | **构成**        |
| ------ | ------------- |
| 现在时    | viene + 过去分词  |
| 过去时    | veniva + 过去分词 |
| 将来时    | verrà + 过去分词  |

**例句**:

1. **Il libro viene pubblicato la prossima estate.** — 这本书将在下个夏天出版。
2. **Il documento è stato firmato dal direttore.** — 文件已被主任签署。
3. **La riunione viene organizzata dalla segreteria.** — 会议由秘书处组织。
4. **Il problema è stato risolto.** — 问题已经解决了。
5. **Il pacco verrà consegnato domani.** — 包裹明天送达。
6. **Il film viene proiettato in tutti i cinema.** — 电影在所有电影院上映。
7. **La lingua viene studiata da milioni di persone.** — 这种语言被数百万人学习。
8. **Il messaggio è stato inviato ieri.** — 信息昨天发送了。

***

#### 6. Il passato remoto（远过去时）

**定义**: Passato remoto（远过去时）用于描述遥远的过去事件，在意大利北部常用，但在南方和书面语中广泛使用。

**构成**:

### 语法点

#### 1. Gli aggettivi in -bile（-bile形容词）

**定义**: 以-bile结尾的形容词有被动含义，表示"可以被..."的可能性。

**构成规则**:

| **动词词根**     | **+ -bile**   | **含义** |
| ------------ | ------------- | ------ |
| legg-ere     | leggibile     | 可读的    |
| cap-ire      | capibile      | 可理解的   |
| cred-ere     | credibile     | 可信的    |
| risolv-ere   | risolvibile   | 可解决的   |
| realizz-zare | realizzabile  | 可实现的   |
| ricicl-are   | riciclabile   | 可回收的   |
| compr-endere | comprensibile | 可理解的   |
| toller-are   | tollerabile   | 可容忍的   |

**否定形式**: 加in-前缀

| **原词**        | **否定词**         |
| ------------- | --------------- |
| credibile     | incredibile     |
| tollerabile   | intollerabile   |
| leggibile     | illeggibile     |
| comprensibile | incomprensibile |
| risolvibile   | irrisolvibile   |

**例句**:

1. **È un'azione realizzabile.** — 这是一个可以实现的行动。
2. **Si tratta di una storia credibile.** — 这是一个可信的故事。
3. **È un materiale riciclabile.** — 这是一种可回收材料。
4. **Il problema non è risolvibile.** — 这个问题无法解决。
5. **È un testo leggibile.** — 这是一篇可读的文章。
6. **La situazione è tollerabile.** — 情况是可以容忍的。
7. **Non è accettabile questo comportamento.** — 这种行为是不可接受的。
8. **È una proposta inaccettabile.** — 这是一个不可接受的提议。

***

#### 2. 接关系从句的关系代词

**详细规则**:

| **关系代词**    | **用法**  | **例句**                                   |
| ----------- | ------- | ---------------------------------------- |
| che         | 主语/宾语   | La persona che parla... 说的人...           |
| cui         | 用介词     | La persona con cui parlo... 我与之说话的人...   |
| il/la quale | 正式/避免歧义 | Il libro del quale parliamo... 我们谈论的书... |

**例句**:

1. **Il libro che sto leggendo è bellissimo.** — 我正在读的这本书很美。
2. **Le persone che parlano italiano sono italiane.** — 说意大利语的人是意大利人。
3. **La ragazza con cui sono uscito è studentessa.** — 我与之约会的女孩是学生。
4. **Il motivo per cui sono venuto è importante.** — 我来的原因很重要。
5. **La casa in cui vivo è grande.** — 我住的房子很大。

#### 1. Le frasi concessive（让步从句）

**定义**: 让步从句表示"尽管...，仍然..."的意思。

#### 1. Il periodo ipotetico dell'irrealtà（虚拟现实条件句）

**例句**:

**构成**: avere/essere的虚拟式未完成时 + 过去分词

**例句**:

#### 3. Periodo ipotetico总结

**三种类型**:

| **类型** | **结构**        | **含义**    |
| ------ | ------------- | --------- |
| 现实条件句  | se + 直陈式现在/将来 | 可能实现      |
| 可能条件句  | se + 虚拟式未完成时  | 不太可能实现    |
| 虚拟条件句  | se + 虚拟式愈过去时  | 不可能实现（过去） |

**例句对比**:

1. **Se studi, superi l'esame.** (很可能) — 如果你学习，你会通过考试。
2. **Se studiassi, superresti l'esame.** (不太可能) — 如果你学习，你会通过考试。
3. **Se avessi studiato, avresti superato l'esame.** (已经过去) — 如果你学习了，你就通过考试了。

#### 16. La forma passiva（被动语态）

**essere被动**:

| **时态** | **构成**      |
| ------ | ----------- |
| 现在时    | è + 过去分词    |
| 过去时    | era + 过去分词  |
| 将来时    | sarà + 过去分词 |

**venire被动**:

| **时态** | **构成**        |
| ------ | ------------- |
| 现在时    | viene + 过去分词  |
| 过去时    | veniva + 过去分词 |
| 将来时    | verrà + 过去分词  |

**例句**:

1. **Il libro è pubblicato.** — 书被出版了。
2. **Il libro viene pubblicato.** — 书正在被出版。
3. **Il libro sarà pubblicato.** — 书将被出版。
4. **Il libro è stato pubblicato.** — 书已经被出版了。
5. **La lingua viene studiata da tutti.** — 这种语言被所有人学习。

#### 17. Il pronome "ne"（ne代词）

**用法**:

6. **Se ne è andato senza dire nulla.** — 他什么都没说就走了。
7. **Non me ne frega niente!** — 我根本不在乎！
8. **Me ne vado!** — 我走了！

#### 18. Il pronome "ci"（ci代词）
