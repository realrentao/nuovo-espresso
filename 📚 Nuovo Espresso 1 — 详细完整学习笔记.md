### 核心对话

#### 1. Ciao o buongiorno?（嗨还是早上好？）— p.5, es.1

**练习说明**: Ascolta e completa i dialoghi.（听录音并补充对话。）

对话场景：

| **意大利语（原文）**                | **中文翻译**              |
| --------------------------- | --------------------- |
| **A**: Ciao, Giorgio!       | **A**: 嗨，Giorgio！     |
| **B**: Ciao!                | **B**: 嗨！             |
| **A**: Buongiorno!          | **A**: 早上好！           |
| **B**: Buongiorno!          | **B**: 早上好！           |
| **A**: Oh, ciao Francesca!  | **A**: 哦，嗨，Francesca！ |
| **B**: Ciao!                | **B**: 嗨！             |
| **A**: Buona sera, dottore! | **A**: 晚上好，医生！        |
| **B**: Buona sera!          | **B**: 晚上好！           |

**时间与问候匹配**（p.5, schema）:

| **时间段**               | **正式问候**        | **非正式问候**          |
| --------------------- | --------------- | ------------------ |
| 早晨 5-13 (mattina)     | **Buongiorno!** | **Ciao! / Salve!** |
| 下午 13-19 (pomeriggio) | **Buongiorno!** | **Ciao! / Salve!** |
| 晚上 19-23 (sera)       | **Buona sera!** | **Ciao!**          |
| 夜间 23-5 (notte)       | **Buonanotte!** | **Buonanotte!**    |

#### 2. Scusi, Lei come si chiama?（请问您叫什么名字？）— p.6, es.2

**练习说明**: Abbina i dialoghi ai disegni. Leggi il dialogo e verifica.（将对话与图片匹配。阅读对话并检查。）

**对话1 — 非正式（tu）**:

> **A**: Ciao, sono Valeria, e tu come ti chiami?\
> **B**: Alberto. E tu?\
> **A**: Io Cecilia.

**对话2 — 正式（Lei）**:

> **A**: Buongiorno, sono Giovanni Muti.\
> **B**: Piacere, Carlo De Giuli.

**对话3 — 正式（Lei），询问姓名**:

> **A**: Scusi, Lei come si chiama?\
> **B**: Franca Cucci.\
> **A**: E Lei?\
> **B**: Anch'io mi chiamo Cucci, Paola Cucci.

**对话4 — 确认身份**:

> **A**: La signora Genovesi?\
> **B**: Sì, sono io, e Lei è il signor...?\
> **A**: Ragazzi. Marcello Ragazzi.

#### 3. Scusa / Scusi — 请对方重复

* **非正式**: Come, scusa?（什么，请再说一遍？）
* **正式**: Come, scusi?（什么，请您再说一遍？）

#### 6. Qual è il tuo numero di telefono?（你的电话号码是多少？）— p.8, es.7

**完整对话**:

> **A**: Qual è il tuo numero di telefono?\
> **B**: Il mio numero di telefono... è 340 154 65 47.\
> **A**: E il cellulare?\
> **B**: 349 25 47 577.\
> **A**: E il tuo indirizzo?\
> **B**: Via Garibaldi 27.\
> **A**: Scusa, qual è il tuo cognome?\
> **B**: De Luca.

**正式版本**:

> **A**: Qual è il Suo numero di telefono?\
> **B**: Il mio numero di telefono... è 340 154 65 47.\
> **A**: E il Suo indirizzo?\
> **B**: Via Garibaldi 27.

> **⚠️ 注意**: **Qual è** 不加撇号！❌ Qual'è → ✅ Qual è

### 语法点

#### 1. 主格代词 (Pronomi soggetto) — p.12

| **人称**   | **意大利语**      | **中文** | **说明**      |
| -------- | ------------- | ------ | ----------- |
| 1ª sing. | **io**        | 我      | 通常省略        |
| 2ª sing. | **tu**        | 你      | 非正式         |
| 3ª sing. | **lui / lei** | 他 / 她  | ​           |
| 尊称       | **Lei**       | 您      | 大写L，用第三人称单数 |
| 1ª plur. | **noi**       | 我们     | ​           |
| 2ª plur. | **voi**       | 你们     | ​           |
| 3ª plur. | **loro**      | 他们/她们  | ​           |

> **注意**:
>
> * 意大利语中主格代词通常省略，因为动词变位已体现人称
> * 只在需强调或对比时使用：**Io** sono di Roma, e **tu**?
> * 尊称Lei（您）也可写作 **Loro**（您们，复数）

#### 2. 动词 essere（是）— 单数现在时 — p.12

| **人称**        | **变位**   | **发音**   |
| ------------- | -------- | -------- |
| (io)          | **sono** | /ˈso.no/ |
| (tu)          | **sei**  | /ˈsɛi/   |
| (lui/lei/Lei) | **è**    | /ˈɛ/     |

**例句**:

1. Io **sono** italiano / italiana. — 我是意大利人。
2. Tu **sei** di Milano? — 你来自米兰吗？
3. Lei **è** la signora Gucci? — 您是Gucci女士吗？
4. Lui **è** il dottor Rossi. — 他是Rossi医生。
5. **È** lui? — 是他吗？

#### 3. 动词 avere（有）— 单数现在时 — p.13

| **人称**        | **变位**  | **发音** |
| ------------- | ------- | ------ |
| (io)          | **ho**  | /ˈɔ/   |
| (tu)          | **hai** | /ˈai/  |
| (lui/lei/Lei) | **ha**  | /ˈa/   |

**例句**:

1. **Ho** un cellulare. — 我有一部手机。
2. **Hai** il numero? — 你有号码吗？
3. La signora **ha** un indirizzo. — 这位女士有一个地址。
4. **Ho** 28 anni. — 我28岁。（正式对话中出现）
5. **Ha** il mio numero? — 他有我的号码吗？

#### 4. 动词 chiamarsi（名叫）— 单数现在时 — p.12

| **人称**        | **变位**        |
| ------------- | ------------- |
| (io)          | **mi chiamo** |
| (tu)          | **ti chiami** |
| (lui/lei/Lei) | **si chiama** |

**例句**:

1. **Mi chiamo** Carlo. — 我叫Carlo。
2. **Ti chiami** Maria? — 你叫Maria吗？
3. Lui **si chiama** Giovanni. — 他叫Giovanni。
4. Lei **si chiama** Paola. — 她叫Paola。
5. Come **si chiama**? — 您叫什么名字？

### 文化角 — Caffè Culturale 1 (p.16)

#### 🏛️ Buongiorno, buona sera o buonanotte?

**意大利原文**:

> Gli italiani salutano in modo molto affettuoso, soprattutto amici e parenti, con baci e abbracci.
>
> **Buongiorno**: dalla mattina fino alle 13 circa.
> **Buona sera**: dalle 19 in poi.
> **Ciao / Salve**: in qualsiasi momento della giornata (informale).
>
> **Con il tu**: arrivederci!
> **Con il Lei**: arrivederla!

**中文翻译**:

> 意大利人问候时非常热情，尤其是朋友和亲戚之间，常用亲吻和拥抱。
>
> **Buongiorno**: 从早晨到大约下午1点。
> **Buona sera**: 从晚上7点以后。
> **Ciao / Salve**: 一天中的任何时间（非正式）。
>
> **非正式告别**: arrivederci!
> **正式告别**: arrivederla!

**文化对比问题**:

1. Nel tuo Paese è normale abbracciare una persona per salutarla? — 在你的国家，问候时拥抱一个人是正常的吗？
2. È una cosa comune tra persone dello stesso sesso come tra persone di sesso diverso? — 这在同性之间和异性之间一样常见吗？

**更多告别表达**:

* Ci vediamo! — 回头见！
* Buona giornata! — 祝你有美好的一天！
* A presto! — 回头见！
* A domani! — 明天见！
* Alla prossima volta! — 下次见！

### 语法点

#### 1. 规则 -are 动词（现在时）— p.24

| **人称**        | **parlare（说）** | **lavorare（工作）** | **abitare（居住）** | **studiare（学习）** |
| ------------- | -------------- | ---------------- | --------------- | ---------------- |
| (io)          | parl**o**      | lavor**o**       | abit**o**       | studi**o**       |
| (tu)          | parl**i**      | lavor**i**       | abit**i**       | studi**i**       |
| (lui/lei/Lei) | parl**a**      | lavor**a**       | abit**a**       | studi**a**       |
| (noi)         | parl**iamo**   | lavor**iamo**    | abit**iamo**    | studi**amo**     |
| (voi)         | parl**ate**    | lavor**ate**     | abit**ate**     | studi**ate**     |
| (loro)        | parl**ano**    | lavor**ano**     | abit**ano**     | studi**ano**     |

> **规则**: 词干 + -o, -i, -a, -iamo, -ate, -ano
>
> **⚠️ 注意 -iare 动词**: studiare, cominciare, mangiare, giocare 等以 -iare 结尾的动词，在 tu 人称中只保留一个 i: tu studi（而非 studii）, tu cominci, tu mangi。而 noi 人称中：noi studiamo, noi cominciamo。

**例句**:

1. Io **parlo** italiano e inglese. — 我说意大利语和英语。
2. Tu **lavori** in un ristorante? — 你在餐厅工作吗？
3. Lui **abita** a Roma. — 他住在罗马。
4. Noi **studiamo** italiano. — 我们学习意大利语。
5. Voi **parlate** molto bene. — 你们说得很好。
6. Loro **abitano** a Milano. — 他们住在米兰。

### 语法点

#### 1. 规则 -ere 动词（现在时）— p.40

| **人称**        | **prendere（拿/取）** | **vedere（看见）** | **scrivere（写）** | **chiedere（问）** |
| ------------- | ----------------- | -------------- | --------------- | --------------- |
| (io)          | prend**o**        | ved**o**       | scriv**o**      | chied**o**      |
| (tu)          | prend**i**        | ved**i**       | scriv**i**      | chied**i**      |
| (lui/lei/Lei) | prend**e**        | ved**e**       | scriv**e**      | chied**e**      |
| (noi)         | prend**iamo**     | ved**iamo**    | scriv**iamo**   | chied**iamo**   |
| (voi)         | prend**ete**      | ved**ete**     | scriv**ete**    | chied**ete**    |
| (loro)        | prend**ono**      | ved**ono**     | scriv**ono**    | chied**ono**    |

> **规则**: -ere 去掉词尾，加 -o, -i, -e, -iamo, -ete, -ono

**例句**:

1. Che cosa **prendete**? — 你们要什么？
2. **Prendo** un caffè. — 我要一杯咖啡。
3. **Vediamo** il menù. — 我们看菜单。
4. **Scrivi** il tuo nome? — 你写你的名字吗？
5. Loro **chiedono** il conto. — 他们要求结账。
6. **Prende** un caffè? — 您要一杯咖啡吗？

#### 2. 规则 -ere 动词补充 — p.40

| **动词**     | **中文** | **用法** |
| ---------- | ------ | ------ |
| prendere   | 拿、取、点餐 | 最常用    |
| vedere     | 看见     | ​      |
| scrivere   | 写      | ​      |
| chiedere   | 问、要求   | ​      |
| leggere    | 阅读     | ​      |
| mettere    | 放、放置   | ​      |
| rispondere | 回答     | ​      |

#### 3. 动词 volere（想要）— 现在时 — p.40

| **人称**      | **变位**       |
| ----------- | ------------ |
| io          | **voglio**   |
| tu          | **vuoi**     |
| lui/lei/Lei | **vuole**    |
| noi         | **vogliamo** |
| voi         | **volete**   |
| loro        | **vogliono** |

> **注意**: volere 是不规则动词，词根变化较大。

**例句**:

1. **Voglio** un caffè. — 我想要一杯咖啡。
2. **Vuole** il menù? — 您要看菜单吗？
3. Che cosa **vuoi**? — 你想要什么？
4. **Vogliamo** prenotare un tavolo. — 我们想预订一张桌子。
5. **Volete** dell'acqua? — 你们要水吗？
6. Loro **vogliono** il conto. — 他们要结账。

**礼貌形式**: 使用 condizionale（条件式）会更礼貌——**vorrei**（我想要）

* **Vorrei** un caffè. — 我想要一杯咖啡。（更礼貌）
* **Vorrei** prenotare un tavolo. — 我想预订一张桌子。

#### 4. 动词 preferire（更喜欢）— 现在时 — p.40

| **人称**      | **变位**           | **中文**   |
| ----------- | ---------------- | -------- |
| io          | **preferisco**   | 我更喜欢     |
| tu          | **preferisci**   | 你更喜欢     |
| lui/lei/Lei | **preferisce**   | 他/她/您更喜欢 |
| noi         | **preferiamo**   | 我们更喜欢    |
| voi         | **preferite**    | 你们更喜欢    |
| loro        | **preferiscono** | 他们更喜欢    |

> **注意**: preferire 属于 **-isco 型** -ire 动词变位，在 io, tu, lui/lei/Lei, loro 的词尾前插入 **-isc-**。

**例句**:

1. **Preferisco** il vino rosso. — 我更喜欢红酒。
2. **Preferisci** la pizza o la pasta? — 你更喜欢披萨还是意面？
3. Lui **preferisce** il caffè. — 他更喜欢咖啡。
4. **Preferiamo** il tè. — 我们更喜欢茶。
5. **Preferite** qui o là? — 你们更喜欢这里还是那里？
6. Loro **preferiscono** la birra. — 他们更喜欢啤酒。

> 更多 -isco 型动词：**finire**（完成）, **capire**（理解）, **pulire**（清洁）, **spedire**（寄送）

| **词**     | **词性**    | **用法** | **例句**                                     |

> **buono 的缩略形式**:

> * **buon**（用于阳性名词前，辅音或元音皆可）: un **buon** libro, un **buon** amico
> * **buona**（阴性）: una **buona** idea
> * **buoni**（阳性复数）: **buoni** amici
> * **buone**（阴性复数）: **buone** amiche

2. Parli molto **bene** l'italiano. — 你意大利语说得很好。
3. È un **buon** vino. — 这是一种好酒。
4. La pasta è **buona**. — 意面很好吃。
5. Oggi sto **bene**. — 我今天很好。

### 语法点

#### 1. 规则 -ire 动词（现在时）— p.53

**类型1：直接变位（dormire型）**

| **人称**        | **dormire（睡觉）** | **partire（出发）** | **aprire（打开）** |
| ------------- | --------------- | --------------- | -------------- |
| (io)          | dorm**o**       | part**o**       | apr**o**       |
| (tu)          | dorm**i**       | part**i**       | apr**i**       |
| (lui/lei/Lei) | dorm**e**       | part**e**       | apr**e**       |
| (noi)         | dorm**iamo**    | part**iamo**    | apr**iamo**    |
| (voi)         | dorm**ite**     | part**ite**     | apr**ite**     |
| (loro)        | dorm**ono**     | part**ono**     | apr**ono**     |

**类型2：-isco 型变位（preferire型）**

| **人称**        | **preferire（更喜欢）** | **finire（完成）** | **capire（理解）** | **pulire（清洁）** |
| ------------- | ------------------ | -------------- | -------------- | -------------- |
| (io)          | prefer**isco**     | fin**isco**    | cap**isco**    | pul**isco**    |
| (tu)          | prefer**isci**     | fin**isci**    | cap**isci**    | pul**isci**    |

| (voi)         | prefer**ite**      | fin**ite**     | cap**ite**     | pul**ite**     |
| (loro)        | prefer**iscono**   | fin**iscono**  | cap**iscono**  | pul**iscono**  |

> **规则**: 在 io, tu, lui/lei/Lei, loro 的词尾前插入 **-isc-**；noi 和 voi 不加。
>
> ⚠️ **注意**: 并非所有 -ire 动词都是 -isco 型。像 dormire, partire, aprire, sentire, seguire 等属于第一种变位类型。

#### 1. C'è / Ci sono（有...）— p.61

| **形式**      | **用法** | **示例**                        |
| ----------- | ------ | ----------------------------- |
| **C'è**     | 单数名词前  | **C'è** la connessione Wi-Fi? |
| **Ci sono** | 复数名词前  | **Ci sono** gli asciugamani?  |

> **注意**: c'è = ci + è（ci 在此处是"那里"的意思，不是"我们"）

**例句**:

1. **C'è** un ristorante qui vicino? — 这附近有餐厅吗？

### 语法点

#### 1. 形容词 (Aggettivi) — 性与数变化 — p.78

**形容词必须与所修饰的名词保持性数一致。**

| **类型**                  | **单数**              | **复数**                  | **示例**                                              |
| ----------------------- | ------------------- | ----------------------- | --------------------------------------------------- |
| -o → -i / -a → -e       | -o (m.) / -a (f.)   | -i (m.) / -e (f.)       | un museo **famoso** / una chiesa **famosa**         |
| ​                       | ​                   | ​                       | musei **famosi** / chiese **famose**                |
| -e → -i                 | -e (m./f.)          | -i (m./f.)              | un monumento **interessante**                       |
| ​                       | ​                   | ​                       | una città **interessante**                          |
| ​                       | ​                   | ​                       | monumenti **interessanti** / città **interessanti** |
| -co → -chi / -ca → -che | -co (m.) / -ca (f.) | -chi (m.) / -che (f.)   | parco **pubblico** / piazza **pubblica**            |
| ​                       | ​                   | ​                       | parchi **pubblici** / piazze **pubbliche**          |
| -ista → -isti / -iste   | -ista (m./f.)       | -isti (m.) / -iste (f.) | un ragazzo **ottimista**                            |
| ​                       | ​                   | ​                       | ragazzi **ottimisti** / ragazze **ottimiste**       |

**形容词位置**:

* 通常位于名词**后面**：una città **tranquilla**, un ragazzo **francese**
* 少数常用形容词可位于名词**前面**：una **bella** città, un **buon** amico, una **grande** piazza
* 当有多个形容词时，顺序灵活：una città **antica e molto bella**

### 语法点

#### 1. 近过去时 (Passato prossimo) — p.98

**构成**: **助动词（avere 或 essere）的现在时 + 过去分词 (participio passato)**

| **助动词**    | **用法**          | **示例**                       |
| ---------- | --------------- | ---------------------------- |
| **avere**  | 大多数及物动词和部分不及物动词 | Ho mangiato. / Ho visto.     |
| **essere** | 大多数表示运动/状态的动词   | Sono andato. / Sono partito. |

**过去分词构成**:

| **动词类别** | **规则** | **示例**                                         |
| -------- | ------ | ---------------------------------------------- |
| -are     | -ato   | parlare → **parlato**, mangiare → **mangiato** |
| -ere     | -uto   | sapere → **saputo**, avere → **avuto**         |
| -ire     | -ito   | dormire → **dormito**, finire → **finito**     |

**完整变位 — parlare（说）**:

| **人称**        | **变位**              |
| ------------- | ------------------- |
| (io)          | **ho parlato**      |
| (tu)          | **hai parlato**     |
| (lui/lei/Lei) | **ha parlato**      |
| (noi)         | **abbiamo parlato** |
| (voi)         | **avete parlato**   |
| (loro)        | **hanno parlato**   |

**完整变位 — andare（去）**:

| **人称**        | **阳性**           | **阴性**           |
| ------------- | ---------------- | ---------------- |
| (io)          | **sono andato**  | **sono andata**  |
| (tu)          | **sei andato**   | **sei andata**   |
| (lui/lei/Lei) | **è andato**     | **è andata**     |
| (noi)         | **siamo andati** | **siamo andate** |
| (voi)         | **siete andati** | **siete andate** |
| (loro)        | **sono andati**  | **sono andate**  |

**助动词选择规则**:

| **使用 avere 的动词**                                           | **使用 essere 的动词**                                             |
| ---------------------------------------------------------- | ------------------------------------------------------------- |

### 语法点

#### 1. 直接宾语代词 (Pronomi diretti) — p.110

| **人称**        | **直接宾语代词** | **含义**   | **示例**                     |
| ------------- | ---------- | -------- | -------------------------- |
| 1ª sing.      | **mi**     | 我        | **Mi** vedi?（你看见我吗？）       |
| 2ª sing.      | **ti**     | 你        | **Ti** chiamo.（我打给你。）      |
| 3ª sing. (m.) | **lo**     | 他/它      | **Lo** prendo.（我拿它/我点它。）   |
| 3ª sing. (f.) | **la**     | 她/它      | **La** compro.（我买它。）       |
| 1ª plur.      | **ci**     | 我们       | **Ci** vede?（他看见我们吗？）      |
| 2ª plur.      | **vi**     | 你们       | **Vi** aspetto.（我等你们。）     |
| 3ª plur. (m.) | **li**     | 他们/它们（阳） | **Li** prendo.（我拿它们/我点它们。） |
| 3ª plur. (f.) | **le**     | 她们/它们（阴） | **Le** compro.（我买它们。）      |

> **注意**: 直接宾语代词通常放在变位动词前面。

**例句**:

1. Il caffè? **Lo** prendo volentieri. — 咖啡？我很乐意喝。
2. La pizza? **La** mangio stasera. — 披萨？我今晚吃。
3. I pomodori? **Li** compro al mercato. — 番茄？我在市场买。
4. Le fragole? **Le** lavo adesso. — 草莓？我现在洗。
5. **Mi** chiami? — 你打电话给我吗？
6. **Ti** aspetto alle 8. — 我8点等你。

**完整形式**（在动词不定式后，需连写）:

* Prendere + lo = **prenderlo**（拿它）
* Comprare + la = **comprarla**（买它）
* Mangiare + li = **mangiarli**（吃它们）
* Mangiare + le = **mangiarle**（吃它们（阴））

**例句**:

1. Voglio **mangiarlo**. — 我想吃它。
2. Puoi **comprarla**? — 你能买它吗？
3. Devo **prenderli**. — 我必须拿它们。

#### 2. 部分冠词 (Articolo partitivo) — p.111

| **形式**    | **用法**              | **示例**                              |
| --------- | ------------------- | ----------------------------------- |
| **del**   | 阳性单数（辅音前）           | **del** pane, **del** formaggio     |
| **dello** | 阳性单数（s+辅音/gn/ps/z前） | **dello** zucchero                  |
| **dell'** | 阳/阴性单数（元音前）         | **dell'** olio, **dell'** acqua     |
| **della** | 阴性单数（辅音前）           | **della** marmellata                |
| **dei**   | 阳性复数                | **dei** pomodori                    |
| **degli** | 阳性复数（元音/s+辅音前）      | **degli** amici, **degli** studenti |
| **delle** | 阴性复数                | **delle** mele                      |

> **含义**: 部分冠词意为"一些/一部分"，相当于英语的"some/any"。
>
> **注意**: 在否定句中部分冠词变为 **di**（不带冠词）：Non ho **di** soldi.（我没有钱。）

**例句**:

1. Vorrei **del** pane. — 我想要一些面包。
2. Ho comprato **della** frutta. — 我买了一些水果。
3. Prendo **dell'** acqua. — 我拿一些水。
4. Ci sono **dei** turisti. — 有一些游客。
5. Mangi **della** pasta? — 你吃些意面吗？
6. Non ho **di** soldi. — 我没有钱。（否定句）

> 在意大利，家庭非常重要。家庭纽带很牢固，子女通常很晚才离开父母家。意大利年轻人离开父母家的平均年龄是30岁。

>
> 近年来，事实婚姻和重组家庭增加了。

> * Il **27%** degli italiani vive vicino alla casa della madre.（27%的意大利人住在母亲家附近。）

> * Il **65%** dei figli maschi fa visita alla madre ogni giorno.（65%的儿子每天探望母亲。）

## 📚 系统语法参考 (Grammatica)

以下是教材第198-230页语法附录的系统整理。

### 1. 定冠词 (Articoli determinativi) — p.206

| **​**  | **单数**     | **复数** |
| ------ | ---------- | ------ |
| **阳性** | il, lo, l' | i, gli |
| **阴性** | la, l'     | le     |

**使用规则**:

* **il** / **i**: 辅音前（il libro → i libri）
* **lo** / **gli**: s+辅音、gn、ps、z前（lo studente → gli studenti）
* **l'** / **gli**: 元音前（l'amico → gli amici, 阳性）
* **la** / **le**: 辅音前（la casa → le case）
* **l'** / **le**: 元音前（l'amica → le amiche, 阴性）

### 2. 不定冠词 (Articoli indeterminativi) — p.207

| **​**         | **阳性** | **阴性** |
| ------------- | ------ | ------ |
| 辅音前           | un     | una    |
| 元音前           | un     | un'    |
| s+辅音/gn/ps/z前 | uno    | una    |

### 3. 名词复数 (Plurale dei sostantivi) — p.207-208

**规则变化**:

* -o → -i: libro → libri
* -a → -e: casa → case
* -e → -i: ristorante → ristoranti
* -ca → -che: amica → amiche
* -ga → -ghe: collega → colleghe
* -co → -chi: cuoco → cuochi（但 medico → medici 例外）
* -go → -ghi: albergo → alberghi（但 psicologo → psicologi 例外）
* -cia/-gia (辅音前) → -ce/-ge: spiaggia → spiagge
* -cia/-gia (元音前) → -cie/-gie: farmacia → farmacie
* 重读结尾/辅音结尾不变: caffè → caffè, bar → bar

### 6. 近过去时 (Passato prossimo) — p.210

**构成**: 助动词（avere/essere）现在时 + 过去分词

**助动词选择**:

| **用 avere**                          | **用 essere**                                        |
| ------------------------------------ | --------------------------------------------------- |
| 大多数及物动词                              | 运动/方向动词（andare, venire, arrivare, partire, uscire等） |
| 部分不及物动词（lavorare, dormire, parlare等） | 状态变化动词（nascere, morire, diventare等）                 |
| ​                                    | 自反动词（alzarsi, chiamarsi, lavarsi等）                  |
| ​                                    | essere, stare, rimanere, piacere 等                  |

**过去分词与主语一致**（essere作助动词时）：

* 阳性单数：-o（andato）
* 阴性单数：-a（andata）
* 阳性复数：-i（andati）
* 阴性复数：-e（andate）

### 9. 间接宾语代词 (Pronomi indiretti) — p.216

| **人称**      | **弱读式（非重读）** | **强读式（重读）**       |
| ----------- | ------------ | ----------------- |
| io          | mi           | a me              |
| tu          | ti           | a te              |
| lui/lei/Lei | gli/le/Le    | a lui/a lei/a Lei |
| noi         | ci           | a noi             |
| voi         | vi           | a voi             |
| loro        | gli          | a loro            |

**例句**:

* **Mi** piace la pizza. — 我喜欢披萨。
* **Ti** telefono stasera. — 我今晚打给你。
* **Le** dico tutto. — 我告诉她一切。
* **Gli** regalo un libro. — 我送他一本书。

### 10. 自反代词 (Pronomi riflessivi) — p.216

| ----------- | ------ |
| io          | mi     |
| tu          | ti     |
| lui/lei/Lei | si     |
| noi         | ci     |
| voi         | vi     |
| loro        | si     |

### 11. 代词 ne — p.217

**用法**:

1. 代替 di + 名词/数量：Quante mele vuoi? **Ne** voglio tre.
2. 表示"从那里"：**Ne** esco adesso.
3. 表示"关于它"：**Ne** parliamo dopo.

### 12. 物主形容词 (Aggettivi possessivi) — p.213

| **人称** | **阳单**      | **阳复**     | **阴单**      | **阴复**      |
| ------ | ----------- | ---------- | ----------- | ----------- |
| 我的     | (il) mio    | (i) miei   | (la) mia    | (le) mie    |
| 你的     | (il) tuo    | (i) tuoi   | (la) tua    | (le) tue    |
| 他/她/它的 | (il) suo    | (i) suoi   | (la) sua    | (le) sue    |
| 您的     | (il) Suo    | (i) Suoi   | (la) Sua    | (le) Sue    |
| 我们的    | (il) nostro | (i) nostri | (la) nostra | (le) nostre |
| 你们的    | (il) vostro | (i) vostri | (la) vostra | (le) vostre |
| 他们的    | (il) loro   | (i) loro   | (la) loro   | (le) loro   |

**冠词使用规则**: 单数亲属名词前不用冠词（mio padre），其余情况用冠词。loro始终用冠词。

### 13. 形容词 — p.212

**一致规则**: 形容词必须与名词的性和数一致。

**位置**: 通常放在名词后，但部分常用形容词可放在前（bello, buono, grande, nuovo等）。

### 14. 绝对最高级 — p.214

| **形式**        | **示例**                               |
| ------------- | ------------------------------------ |
| molto + 形容词   | **molto** bello                      |
| -issimo/a/i/e | bell**issimo**, bell**issima**       |
| 重复形容词         | piano **piano**, sottile **sottile** |

### 15. 副词构成 — p.214

| **规则**         | **示例**                                                 |
| -------------- | ------------------------------------------------------ |
| 阴性形容词 + -mente | libera + mente = **liberamente**                       |
| -e结尾直接加        | elegante + mente = **elegantemente**                   |
| -le/-re去e加     | normale → **normalmente**, regolare → **regolarmente** |
| 不规则            | **bene**（好的）, **male**（坏的）, **molto**（很）               |
